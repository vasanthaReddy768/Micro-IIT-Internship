# Micro-IIT-Internship
The To-Do List project is a web application built using HTML, CSS, and JavaScript, allowing users to add, delete, and manage tasks efficiently.
The Rock-Paper-Scissors project is a simple game where the user competes against the computer by selecting rock, paper, or scissors. Built with HTML, CSS, and JavaScript, it features randomized computer choices, real-time result display, and interactive UI updates, demonstrating basic game logic and frontend development skills.
